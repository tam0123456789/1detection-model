# Trafic: Vietnamese Traffic Sign Detection with YOLO11n and YOLO11n-UIB

This repository presents a traffic sign detection project for Vietnamese real-world traffic signs using:

- YOLO11n baseline
- YOLO11n-UIB with a custom UIB backbone
